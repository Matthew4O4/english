## 强调副词

### 强调形容词或副词

| completely | 完全地，彻底地                                               |
| ---------- | ------------------------------------------------------------ |
| perfectly  | 完美地，极佳地；绝对地，完全地                               |
| really     | 真正地，事实上；确实的，的确（用于加强语气）；很，十分；（用于带有隐含的否定回答的疑问句和感叹句）真的，是吗；用于缓和否定语气 |
| totally    | 完全地，整个地；对极了，太对了                               |
| mainly     | 主要地，首要地；大部分，大多                                 |
| only       | 只，只有；仅在……情况下（或地方等）；只是，不过（表示不重要、不严重等）；（强调数量少或时间段）仅仅，才；不早于，直到……才；（除此之外别无可为）只能；（用于说明事情的恶果）只会；不料，竟然；只能（希望或愿望等）；（强调行动恰当）完全，真正；刚刚（强调某事刚发生）；险些没，差点没 |
